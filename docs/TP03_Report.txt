1. PROJECT OVERVIEW
-------------------

Workout Log is a console-based Java application for creating, viewing, editing, deleting,
searching, and persisting workout entries. The codebase maintains clear separation of concerns:

- Workout.java      : Data model with getters/setters and CSV conversion helpers
- WorkoutLog.java   : In-memory workout list management (add, remove, edit, search)
- WorkoutApp.java   : CLI menu, input validation, and application flow
- FileStorage.java  : CSV save/load, validation, backup rotation, and thread-safe file I/O
- User.java         : Simple user identity class (username)
